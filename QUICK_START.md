# Quick Start - Environment Variables Page

## 3 Steps to Get Started

### 1. Install Dependencies
```bash
npm install @radix-ui/react-tabs @radix-ui/react-alert-dialog
```

### 2. Run SQL Migration
Copy and run `database/migrations/migration-password-view-otps.sql` in Supabase SQL Editor

### 3. Start Dev Server
```bash
npm run dev
```

## Test It

1. Go to `http://localhost:3000/dashboard/environment`
2. Click "Add Variable"
3. Add a test variable
4. Click Eye icon to reveal value
5. Check console for OTP (6-digit code)
6. Enter OTP to reveal value
