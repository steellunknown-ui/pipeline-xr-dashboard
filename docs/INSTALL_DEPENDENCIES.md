# Install Missing Dependencies

Run these commands to install the required Radix UI packages:

```bash
npm install @radix-ui/react-tabs @radix-ui/react-alert-dialog
```

Or if you use yarn:

```bash
yarn add @radix-ui/react-tabs @radix-ui/react-alert-dialog
```

Or if you use pnpm:

```bash
pnpm add @radix-ui/react-tabs @radix-ui/react-alert-dialog
```
